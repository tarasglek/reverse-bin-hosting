# Reverse Bin v0.0.5

## Highlights

- Static HTML applications now use managed Unix sockets instead of loopback TCP ports.
- Executable applications share a consistent unrestricted outbound-network policy.
- Chromium-based applications can read sysfs while retaining the rest of the sandbox.

## Breaking changes

None. Existing application directories and configuration files remain compatible.

## Full list of changes

- Bundle reverse-bin-detector v0.1.12 with static Unix-socket routing and executable sandbox fixes.
- Update the packaged runtimes to uv 0.11.29 and Deno 2.9.3.
- Document static application sockets and the revised security posture.
