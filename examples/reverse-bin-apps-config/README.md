# Deprecated bundle example

This directory documents the old bundle-based deployment flow and is no longer the recommended installation path.

Use the Debian package instead:

- binary: `/usr/bin/reverse-bin-caddy`
- config: `/etc/reverse-bin/Caddyfile`
- apps: `/var/lib/reverse-bin/apps/`
- examples: `/usr/share/doc/reverse-bin/examples/`
